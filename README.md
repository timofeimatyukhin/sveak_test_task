# SVEAK Test Task

## Стек

- HTML
- SCSS
- JavaScript
- Vite

JavaScript используется только для динамической смены количества карточек в демо-режиме.

## Что реализовано

- Адаптивный layout для mobile, tablet, laptop и desktop.
- Sticky header.
- Sidebar с открытием и закрытием без JavaScript.
- Анимация burger-menu.
- Адаптивная сетка карточек.
- Скрытие неполного последнего ряда карточек.
- Динамическая смена количества карточек для проверки сетки.
- Защита текста от переполнения.
- Hover- и focus-состояния.
- Учет `prefers-reduced-motion`.
